Hexo和Github搭建自己的博客

1、环境的搭建

1、Node.js的安装

  • 建议到Node.js官网下载稳定版本的Node.js

  • 安装完成之后，检查一下是否安装成功。分别在命令行界面执行下面两个命令，如果打印出版本号则说明安装成功。

    • node -v
    • npm -v

2、git的安装

git官网下载地址

安装成功后执行下面命令行,打印出git版本号，即说明安装成功

  • git –version

3、Github账户

  • Github官网注册登陆后新建一个代码仓库
  • Repository name 建议取为 yourname.github.io
  • 接下来开启github-page功能，点击项目主页右边的Setting按钮，向下找到GitHub Pages,如下图：

在Select Source选项栏可以看到GitHub pages 现在是Disable的，我们选择master branch，点击Save 来启用 GitHub pages功能。

  • 在浏览器里键入 yourname.github.io 你可以看到GitHub替你创建出的pages页面。
  • 到此，GitHub这边的配置就已经准备完毕了。如果 yourname.github.io 不能访问，注意一下yuorname和gitgub的名字是不是不一样，如果不一样的话，那么你的链接应该是 githubname.github.io/yourname.github.io 建议将两个名字保持一致。

2、安装Hexo

  • 选择一个合适的地方，创建一个文件夹，这里我以 D:/Hexo 为例
  • 打开命令行窗口，cd 到 D:/Hexo
  • 键入下面命令
    • npm install hexo-cli -g
  • 等待一段时间后你将会看到如下图结果：

  • 然后键入

    • npm install hexo –save

  • 等执行完成以后，Hexo就安装完成了。我们检查一下

    • hexo -v
  • 命令行窗口刷出如下图一些信息，就说明安装成功了

3、配置Hexo

  • 我们在刚刚的Hexo文件夹下再创建一个文件夹，我以blog为例
  • 命令行窗口 cd 到该 blog 文件夹下
  • 然后执行下面命令
    • hexo init
    • npm install
  • hexo init 的时候中间可以回出现fatal: unable to access 询问是否重试，可以键入y回车重试几次，不行的话就键入n，回车， git clone failed. Copying data instead。用clone的方式替代下载的方式。
  • 相关配置就完成了，我们来体验一下Hexo的效果，键入：
    • hexo g
  • 在浏览器中打开 http://localhost:4000/ 你将会看到：

  • 到此，本地 Hexo 的工作就已经做完了。

    4、将Hexo和Githug pages 联系起来

  1. git 环境的配置
  • 设置Git的user信息
    • git config –global user.name “guohongxin”
    • git config –global user.email “love_tomatoo@163.com“
  • 生成秘钥
    • ssh-keygen -t rsa -C “love_tomatoo@163.com“
  • github配置公钥

打开github的setting 进入Setting页面如下图：

点击SSH and GPG keys,然后点击 New SSH key，将生成的pub_key用记事本打开，复制到key的输入框内，点击 Add SSH key即可。

  1. Deployment的配置
  • 在D:/Hexo/blog(你自己创建的地方)文件夹下找到 _config.yml 文件，打开后拉到最下方，如下图：

把下面内容copy进去

  • 将repo： 替换成自己创建的yourname.github.io的 SSH 链接即可，地址在如图位置

  • 至此，Hexo和Github pages的连接就建立完成了。

5、发布文章

  • 新建一篇文章
    • hexo new post “hello blog”

  • 在 D:\Hexo\blog\source_posts 下就可以看到 hello-blog.md 文件了。我们可以就用md编辑器写自己的博客了。
  • 博客写完以后，就可以生成部署了
    • hexo g // 生成
  • hexo d // 部署

  • 执行 hexo d 的时候，如果报出 ERROR Deployer not found: git 那么执行下面命令行, 再执行 hexo d 即可

    • npm install hexo-deployer-git –save

  • 这时候，就可以在 yourname.github.io 看到你的博客了。
